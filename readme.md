## Overview

 a simple command line note taking app. The app  allows users to create, edit and delete notes. The app  also allow users to search for notes by title and content.

## Feature

- create note
- edit note
- display note contents
- delete note
- Search for a note


## Used Packages
  ansicolor: ^2.0.2
  dolumns: ^2.0.1
